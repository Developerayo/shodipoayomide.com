## Usage

Run locally:

Clone
```
git clone https://github.com/Developerayo/shodipoayomide.com.git
```

Move into the file with
```
cd shodipoayomide.com
```

Then
```
yarn install
```
Start dev server

```
yarn dev
```
now open https://localhost:8000.

## Installing gatsby-cli

```
npm i gatsby-cli
```
